CodeHSProxy Guide

Follow these steps in order. Take your time and read each step carefully before proceeding.

1. Go to CodeHS Teacher Signup

2. Fill in details (they do not verify email)

3. Go toCodeHS Sandbox

4. Make a valid username

5.Click Switch to code view and paste the contents of the attached html file

6. Click the 3 lines to the left (Files)

7. Make 2 new files, bareworker.js and sw.js, and paste the contents of the respective attatched files in.

8. Click See homepage in new window

9. If step 8 is blocked, simply press Run Code, and stretch the preview window to use the proxy.
